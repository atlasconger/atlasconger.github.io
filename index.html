So You Want to Start Coding

...with Python on a Windows 10 machine, maybe an old chromebook and absolutely zero base knowledge

STEP ONE

Go to Python.org, don't get scared by all of the jargon, and head straight to Downloads--> Windows

STEP TWO

Find the header that says "Stable Releases" and click on the Windows insaller (64-bit) INSERT EXPLANATION HERE

STEP THREE

Once the installer is downloaded, launch it. Follow the intructions inside the installer, you're good to leave everything at default settings.

STEP FOUR

We're going to be using Anaconda for this tutorial. EXPLANATION: Anaconda is a "distrubution" of Python used specifically for data science and machine learning applications. Go to the Anaconda Downloads Website and download the 64 bit grpahical installer.
